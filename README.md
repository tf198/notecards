# NoteCards

Simple one-page game for learning your string fingers.

No installation - just clone it to your webserver or run locally

```bash
git clone https://github.com/tf198/notecards
firefox notecards/index.html
```

## Features

* [x] Basic Violin mode
* [x] Basic Viola mode
* [ ] Basic Cello mode
* [ ] All fingers
* [ ] Note names
